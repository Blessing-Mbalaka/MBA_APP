❌ TROUBLESHOOTING:

  Error: "No module named 'django'"
  → Fix: pip install -r requirements.txt

  Error: "Can't find manage.py"
  → Fix: cd to correct directory (where manage.py is)

  Error: "Permission denied" (Windows)
  → Fix: Run as Administrator

  Test runs but all pages fail
  → Check: Is Django server still running? Stop it first.

  More help: Read QUICK_START.md → Troubleshooting section
